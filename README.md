# PowerUp

Мобильное приложение-таймер для домашних тренировок без инвентаря. Проводит пользователя через последовательность из 6 упражнений с обратным отсчётом, голосовыми подсказками и метрономом.

![images](img/powerUp.jpg)

## Упражнения

| # | Упражнение      | Длительность |
|---|-----------------|--------------|
| 1 | Отжимания       | 20 сек       |
| 2 | Планка          | 90 сек       |
| 3 | Боковая планка  | 60 сек       |
| 4 | Альпинист       | 50 сек       |
| 5 | Прыжки          | 50 сек       |
| 6 | Приседания      | 20 сек       |

Общее время активной работы — **4 минуты 50 секунд** (плюс голосовые объявления между упражнениями).

## Возможности

- Обратный отсчёт для каждого упражнения с автоматическим переходом к следующему
- Голосовые объявления перед каждым упражнением
- Метроном на протяжении всей тренировки для поддержания темпа
- Звуковой сигнал за 5 секунд до конца упражнения
- Сигнал смены стороны на 33-й секунде боковой планки
- Кнопка «Готово» для досрочного перехода к следующему упражнению
- Иллюстрации упражнений в стиле стикмен
- Тёмная тема с бирюзовым акцентом

## Требования

- Node.js 20+
- npm
- Android SDK (для локальной сборки)
- Python 3 + Pillow (только для генерации изображений)

## Запуск

```bash
npm install
npx expo start
```

Для запуска на устройстве или эмуляторе:

```bash
npx expo run:android
```

## Сборка APK

### Локальная сборка (Gradle)

```bash
build-android.bat
```

APK появится в `android/app/build/outputs/apk/release/app-release.apk`.

### Сборка через EAS

```bash
eas build --platform android --profile preview
```

Оба профиля (`preview` и `production`) собирают APK-файл.

## Структура проекта

```
PowerUp/
├── app/                     # Экраны (файловая маршрутизация)
│   ├── _layout.tsx          # Корневой layout (Stack, тёмная тема)
│   ├── index.tsx            # Главный экран — старт тренировки
│   ├── workout.tsx          # Экран тренировки — таймер, звуки, переходы
│   └── finish.tsx           # Экран завершения
├── components/              # UI-компоненты
│   ├── Button.tsx           # Кнопка (primary / secondary)
│   ├── ProgressBar.tsx      # Прогресс-бар «Упражнение X из 6»
│   ├── TimerDisplay.tsx     # Отображение таймера (M:SS)
│   └── RepCounter.tsx       # Счётчик повторений (зарезервирован)
├── hooks/                   # Хуки
│   ├── useTimer.ts          # Обратный отсчёт
│   └── useSound.ts          # Загрузка и воспроизведение звуков
├── constants/               # Константы
│   ├── exercises.ts         # Список упражнений
│   └── theme.ts             # Цвета, отступы, размеры шрифтов
├── utils/
│   └── formatTime.ts        # Форматирование секунд в M:SS
├── assets/
│   ├── images/              # Иллюстрации упражнений (PNG)
│   └── sounds/              # Голосовые объявления, метроном, сигналы
├── generate_exercises.py    # Генератор иллюстраций (Python + Pillow)
├── generate_side_plank.py   # Генератор иллюстрации боковой планки
├── app.json                 # Конфигурация Expo
├── eas.json                 # Конфигурация EAS Build
└── tsconfig.json            # Конфигурация TypeScript
```
